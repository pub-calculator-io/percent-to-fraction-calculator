The percent to fraction calculator converts the given percentages into fractions. If the percent value exceeds 100, the calculator performs percent to mixed number conversion.

[https://www.calculator.io/percent-to-fraction-calculator/](https://www.calculator.io/percent-to-fraction-calculator/)

== Usage ==

Add the Percent to Fraction Calculator shortcode to your page, post or sidebar:

`[ci_percent_to_fraction_calculator]`

Add the following code to your template where you would like the Percent to Fraction Calculator to appear:

`<?php display_ci_percent_to_fraction_calculator(); ?>`

== Installation ==

1. Upload the Percent to Fraction Calculator /ci_percent_to_fraction_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Percent to Fraction Calculator plugin through the "Plugins" menu in WordPress.
